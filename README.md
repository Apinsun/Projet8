# 📊 API de Scoring de Crédit - Prêt à Dépenser

Ce projet contient l'API de prédiction pour le calcul du score de probabilité de défaut de paiement d'un client. Ce modèle a été entraîné avec un algorithme LightGBM et est versionné avec MLflow.

## 🚀 Accès à la documentation de l'API

L'API est construite avec FastAPI. Vous pouvez tester les différents points d'entrée (endpoints) et consulter la documentation interactive via Swagger UI ici :
